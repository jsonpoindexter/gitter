# gitter
Interactive terminal for easily searching and cloning GitHub repositories using Kotlin and ghq.

### Build/Run:
* `./gradlew build`
* `java -jar ./build/libs/gitter-1.0-SNAPSHOT.jar`

### Dependencies:
* ghq - https://github.com/motemen/ghq
